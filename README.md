## Proyecto destacado

### [H&M Fashion Recommendations — FashionData Insights](https://github.com/samucass07/PF_FashionDataInsights)

Sistema de recomendación personalizada construido sobre **31M+ transacciones** y **1.37M clientes**.

- 3 modelos: Popularidad por generación · Filtrado colaborativo · Híbrido
- Pipeline orquestado con Apache Airflow · Contenerizado con Docker
- Dashboard en Power BI · App en Streamlit + FastAPI
- Evaluación con MAP@12, NDCG@12, Precision y Recall
